# Bootstrap

This workspace was created by Neon Agent Framework onboarding.

Run:

```bash
neon doctor
neon blueprint
neon run "Say hello"
neon memory add --title "First memory" --body "My agent is installed."
neon memory search "installed"
neon task add "Create first local task"
neon start --once
```

Configure channels in `~/.neon-agent/config.json`.
