# Starter Agent

This workspace is generated by `neon onboard`.

## Commands

```bash
neon doctor
neon blueprint
neon run "Summarize this workspace"
neon memory add --title "First memory" --body "This workspace belongs to me."
neon memory search "workspace"
neon task add "Create my first owned agent memory"
neon start
```

## Operating Defaults

Read these before wiring a real channel:

- `OPERATING.md` for task triage, updates, approvals, and handoffs.
- `channels/discord-operating-floor.md` for Discord-style chat behavior.
- `agents/handoff-contract.md` when another agent reviews or continues a slice.

## First Task

Send a message to the agent runtime:

```bash
neon run "Say hello from my new agent"
```

Then wire a real channel through your local config.
